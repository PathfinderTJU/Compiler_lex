### 运行方法

- 测试运行环境：Ubuntu 14.04 LTS

- 建议运行环境：Linux操作系统

- 运行命令：

  ~~~
  cd compile文件所在目录下
  compile [文件名]
  ~~~

  - 文件名必须放在与compile同一目录下
  - 必须至少有一个文件名，否则会报错，同一文件可以在一次命令中被识别多次，不存在的文件将显示错误信息，但不会影响之后的文件
